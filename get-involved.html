Thingweb

Thingweb node-wot is a runtime and tooling for web technology
that enables easy creation of applications for the Internet of Things

Get involved

There are various ways how you can get involved including contributions on GitHub :
https://github.com/eclipse/thingweb.node-wot/

  • Create and contribute your own protocols bindings besides HTTP, CoAP, Websockets, ...
  • Build your WoT solutions and share them
  • Report any issue you encounter (and even better provide already a fix)
